⭐ Opzione 1 — Online (File Manager)
Modifica i file direttamente dal browser senza installare nulla. Adatta per modifiche rapide o per chi non ha un computer personale configurato.
🖥️ Opzione 2 — Computer di Scuola (VSCode + Cartella Remota)
Dai PC scolastici connetti al dominio, la cartella remota viene montata come cartella locale. Installa l'estensione VSCode per aprirla nell'editor.
🔒 Opzione 3 — Computer Personale (VSCode + SSH)
Da casa o da qualsiasi PC con accesso SSH al server, usa l'estensione Remote - SSH di VSCode per modificare i file direttamente sul server.
📦 Opzione 4 — Computer Personale (PHP Locale + Deploy)
Installa PHP localmente, sviluppa sul tuo PC, poi carica i file con lo script deploy.sh:

./deploy.sh nome.cognome
